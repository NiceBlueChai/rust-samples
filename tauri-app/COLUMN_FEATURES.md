# Bug列表自定义列功能

## 功能简介
Bug列表现在支持自定义列显示功能，用户可以根据需要选择显示或隐藏特定的列。

## 使用方法

### 1. 打开列设置
- 在Bug列表页面右上角，点击设置图标（齿轮图标）
- 会弹出"列设置"对话框

### 2. 选择显示列
- 在对话框中，您会看到所有可用的列选项
- 勾选您想要显示的列
- 取消勾选您想要隐藏的列

### 3. 保存设置
- 点击"保存"按钮保存您的设置
- 设置会自动保存到浏览器本地存储中
- 下次打开应用时会自动应用之前的设置

### 4. 重置设置
- 点击"重置"按钮可以恢复所有列的默认显示状态

## 技术实现

### 数据结构
```typescript
interface ColumnConfig {
  key: string;        // 列的唯一标识
  label: string;      // 列显示名称
  width?: number;     // 固定宽度
  minWidth?: number;  // 最小宽度
  fixed?: string;     // 固定位置 ('left' | 'right')
  visible: boolean;   // 是否可见
}
```

### 本地存储
设置保存在 `localStorage` 中，键名为 `bugListColumnSettings`

### 响应式更新
使用 Vue 3 的 `computed` 属性实现响应式列显示，当设置改变时表格会自动更新。

## 注意事项
- 操作列始终显示，不可隐藏
- 至少需要显示一列才能正常使用
- 列设置仅影响当前浏览器，不会同步到其他设备
